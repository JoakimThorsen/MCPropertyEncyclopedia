Propertydle - A Minecraft Block Guessing Game

Yes. This is a thing now.

Your goal is to find the solution by guessing different Minecraft blocks and getting clues based on which properties match the target block.

If the cell is Green, then this property's value matches that of the solution.

If it is Red, then this is not the case.

If the solution has multiple values for this property, then each correct value will be marked Yellow, until you get them all right.

For numbers, the color denotes whether your guess is Lower or Higher than the answer, or if your guess is somewhere in between the values of the answer.
